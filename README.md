# Training Planner

## Автор
Криницын Дмитрий Евгеньевич

## Описание
GUI-приложение для планирования тренировок с возможностью:
- Добавления тренировок (дата, тип, длительность)
- Фильтрации по типу и дате
- Сохранения данных в JSON формате
- Валидации ввода

## Требования
- Python 3.x
- tkinter (встроен в Python)

## Установка зависимостей
Дополнительные зависимости не требуются. Все необходимые библиотеки входят в стандартную библиотеку Python.

## Как запустить
```bash
python main.py
```

## Примеры использования
1. Запустите приложение
2. Введите дату в формате YYYY-MM-DD
3. Укажите тип тренировки
4. Введите длительность в минутах
5. Нажмите "Добавить тренировку"
6. Используйте фильтры для поиска

## Пример файла trainings.json
```json
[
  {
    "date": "2024-01-15",
    "type": "Бег",
    "duration": 30
  },
  {
    "date": "2024-01-16",
    "type": "Плавание",
    "duration": 45
  }
]
```
